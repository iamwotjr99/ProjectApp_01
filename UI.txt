*색깔 : 완성된 UI에 각자 원하는 색깔 넣기(보류)

*로딩 화면 : 이재석 친구분께 부탁

*구성 : 친구 목록(최영리) + 탭(양윤기) + 채팅방(양윤기)
챗봇은 탭에 넣어서 구성하기 (탭 기능 : 친구 목록 + 채팅 목록 + 챗봇 목록 + 설정)
친구 추가 : 연락처 동기화 + (+ 버튼)으로 추가하기

*채팅 목록 : 시간순, 개인/단체 구분 없이, 스택 한 번은 토스트로 최근에 보낸 메세지 보여주기, 스택 두 번은 채팅방 들어가기
단체 채팅방의 프로필은 최대 4명의 프로필 보이기

*채팅방 : 이름 + 프사 + 말풍선 + 보낸 시간 + 미디어 보내기 버튼(+ 버튼) + 메세지 보내기 버튼

*채팅방_햄버거 : 옆으로 땡기기 + 버튼 누르기
햄버거 구성 : 친구 목록 + 친구 추가 + 채팅방 나가기

*단체 채팅방 : 친구 구분은 이름과 프사로 구분하기, 위쪽에 참가자 수 보여주기

*미디어 자료 보내기
두 가지 버튼(촬영 + 보내기), 채팅방 내부에서 사진첩 보여주기(실패시 intent로 다른 창 보여주기)

*챗봇 : 챗봇의 주제를 선택할 수 있게 처음부터 채팅창 밑쪽에 선택지 보여주기 + 명령어 인식하면 지정된 말 보내기

*가계부 : 챗봇이 선택지를 먼저 제시, 잘못된 정보 입력시 사용자가 직접 정보 수정할 수 있게 만들기

*유튜브 : 태그어를 입력하면 그 태그어에 맞는 유튜브 url 제시, 챗봇이 장르별로 선택지 제시

*설정 : 리스트뷰로 목록 보여주기 + 하나의 목록 선택하면 다른 목록은 밑으로 밀리고 자세한 목록은 그리드로 표현하기
